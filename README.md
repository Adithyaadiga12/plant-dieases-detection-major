# AgroVision AI

A modern, offline-first Flutter application for plant disease detection using TensorFlow Lite.

## ✨ New Features

- **🧠 Enhanced AI Logic:**
  - **Edge Case Handling:** Real-time detection of blur, dark/overexposed images, and non-leaf content.
  - **Robustness:** 38-class classification with confidence thresholding to prevent false diagnoses.
  - **Safety:** Automatic suppression of severity metrics for healthy plants.
- **🎨 Modern Aesthetic UI:**
  - **Smooth Animations:** Staggered grid entries, breathing micro-interactions, and Hero transitions.
  - **Fluid UX:** Modernized result cards with clearer hierarchy and detailed diagnostic info.
  - **Material 3:** Optimized for the latest Android design standards with dark mode support.
